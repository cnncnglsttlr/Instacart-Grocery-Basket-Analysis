Step 5: Create a Profiling Variable

In[22]:
def profile_segment(row):
    if row['dependants'] >= 2 and row['income'] > 120000:
        return 'Parent, High Income'
    elif row['dependants'] >= 2:
        return 'Parent, Low/Med Income'
    elif row['dependants'] == 0 and row['Age'] < 35:
        return 'Young Adult, No Kids'
    elif row['dependants'] == 0 and row['Age'] >= 35:
        return 'Older Adult, No Kids'
    else:
        return 'Other'

# Apply to df_active
df_active['profile_segment'] = df_active.apply(profile_segment, axis=1)

Step 8: Insights from Crosstab and Heatmap

The heatmap visualizes the distribution of customer profile segments across different U.S. regions.

  • The South has the highest number of customers in the "Parent, High Income" category.
  • The West shows a balanced distribution, especially in the "Young Adult, No Kids" group.
  • The Midwest appears to have relatively fewer "High Income" customers, suggesting a potential opportunity for targeting.
  • The Northeast has a moderate number of "Other" profiles, which may require further segmentation.

This information can be useful for tailoring marketing strategies and customer outreach based on regional characteristics.
